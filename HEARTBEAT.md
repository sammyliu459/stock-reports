# Heartbeat Routine

Rotate through proactive tasks. Don't do everything every time - pick based on day and what's needed.

## Daily Check (every heartbeat)
- If 12+ hours since last Moltbook check: check Moltbook feed/DMs and Hacker News top stories, notify human in Telegram group "-1003862869671" topic "24" if anything interesting or relevant to current interests (AI, Markets, Tech)

## Rotating Proactive Tasks

### Monday/Wednesday/Friday: System Maintenance
- Look for old/temp files to clean up
- Check for config improvements
- Review pending tasks or stale projects
- Fix small friction points

### Tuesday/Thursday: Memory Hygiene
- Review recent `memory/YYYY-MM-DD.md` files
- Update `MEMORY.md` with significant learnings
- Archive outdated information
- Ensure important context is written down

### Saturday: Security & Skills Review
- Audit installed skills - do I still need them?
- Check for skill updates
- Review ClawdHub for useful new skills (with security awareness)
- Clean up unused dependencies

### Sunday: Weekly Reflection
- What worked well this week?
- What mistakes did I make? (update self-review.md)
- Any improvements to my practices?
- Update SOUL.md or AGENTS.md if needed

## Self-Review File
Track mistakes and lessons in `memory/self-review.md`. Write it down immediately when I mess up.

## Quiet Hours
Stay silent (HEARTBEAT_OK) during 23:00-08:00 local time unless urgent.
