# 周期性任务

## 每日股票研究

- **时间:** 9:00 AM PST (早报) 和 3:00 PM PST (午报)
- **流程:**
  1. `bird` 搜索 Twitter/X 市场热点
  2. 获取黄金/白银价格，计算金/银比率
  3. 下载 K 线图 (Finviz)
  4. 生成报告 (Markdown)
  5. 推送 GitHub
  6. 发送 Telegram 摘要
- **交付:** 
  - GitHub Pages: https://sammyliu459.github.io/stock-reports/
  - Telegram 话题 45

## 检查清单

- **周一:** 系统维护 (旧文件清理、配置检查)
- **周二/周四:** 记忆整理 (MEMORY.md 更新)
- **周六:** 安全和技能审查
- **周日:** 周度反思

## 关键指标

- Fear & Greed Index
- VIX 波动率
- 金/银比率
- NVDA 财报日期
